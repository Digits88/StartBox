# sbx_get_term_filter

## Description

Used on the `get_term` filter to add and include custom meta.

> You will probably never need to call this function directly.

## Usage

	<?php sbx_get_term_filter( $term ); ?>

## Parameters

* **term**

	(object) (required) Term object to filter
